# Prestige Catering and Events - Website

## Project Overview
This is the official website for Prestige Catering and Events (Pty) Ltd, 
a professional catering company based in Pretoria, Gauteng.

Built as part of WEDE5020 Portfolio of Evidence.

## Pages
- index.html — Home page
- about.html — About us page
- products.html — Services page
- gallery.html — Gallery page
- enquiry.html — Enquiry form page
- contact_us.html — Contact form page

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Google Fonts (Montserrat, Open Sans)
- Google Maps Embed API

## How to Run
Open index.html in a browser or use Live Server in VS Code.
